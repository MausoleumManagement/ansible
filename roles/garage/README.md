# Ansible Role: Garage

Deploys [Garage](https://garagehq.deuxfleurs.fr/), an open-source distributed
S3-compatible object storage service, to Debian/Ubuntu hosts using systemd.

Based on the official documentation:
- [Quick Start](https://garagehq.deuxfleurs.fr/documentation/quick-start/)
- [systemd setup](https://garagehq.deuxfleurs.fr/documentation/cookbook/systemd/)
- [Ansible cookbook](https://garagehq.deuxfleurs.fr/documentation/cookbook/ansible/)
